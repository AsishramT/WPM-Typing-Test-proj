# WPM Typing Test

A typing speed test desktop app built with Python and Pygame.

## Features
- Random sentence generator
- Real-time WPM (words per minute) calculation
- Accuracy tracking
- Reset functionality

## Requirements
- Python 3.x
- Pygame

## How to Play
1. Launch the app
2. Click the input box to start the timer
3. Type the sentence shown on screen as fast and accurately as you can
4. Press Enter when done to see your results
5. Click the reset icon to play again

## Results
After pressing Enter, the app displays:
- Time — how long it took in seconds
- Accuracy — percentage of correctly typed characters
- WPM — your words per minute score
